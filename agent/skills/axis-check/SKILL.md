# Axis Check

Check only observable ownership boundaries:

```text
Brick = work contract / required return shape / write scope
Agent = performer / received work / returned evidence
Link = Movement / target / handoff / carry refs
```

Return matched, missing, mismatched, or unknown observations. Do not choose
Movement, route target, success, failure, approval, or quality.
