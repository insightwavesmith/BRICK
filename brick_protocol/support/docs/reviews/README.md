# Review Evidence

Review files in this directory are support evidence only.

After FSR-0, active review records live under `brick_protocol/support/docs/reviews/`.
Historical records may mention the old `docs/reviews/` path as historical
evidence. Review PASS output never creates source truth, success judgment,
Movement authority, commit permission, or push permission.

They are not:

```text
source truth
success judgment
owner axis
Movement authority
production readiness
```

once enumerated moved to the archive museum with the rest of the unpinned
historical evidence and no longer resolve here (verified: 0/49 of the old
listed filenames exist on disk). That dead enumeration was trimmed as
confirmed-stale residue. Active review records are whatever `.md` files
currently sit in this directory.
