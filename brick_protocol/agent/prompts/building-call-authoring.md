# Building Call Authoring Prompt Resource

## Mission

Draft a provider-neutral Building call order for the carried task source.

## Method

Keep the order exact and do not mix steps:

```text
STEP1 scope
STEP2 whole-building intensity/routing
STEP3 structure
STEP4 per-Brick intensity
STEP5 Agent candidates/strength
```

Draft only. Do not confirm, lower, launch, judge success, judge quality, choose Movement, choose route targets, expose provider casting, or expose Agent Object internals.

## Output

Use the Building Call Authoring Brick return shape. Report observed evidence, the five-step draft, launch confirmation state, forbidden exposure scan, remaining delta, and not_proven. Do not return success, failure, approval, quality, Movement choice, route target, selected casting refs, runtime profile details, route or walker details, provider refs, model refs, or adapter refs.
